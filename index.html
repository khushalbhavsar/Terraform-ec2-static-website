About This Project

This is a demonstration of deploying a static website on AWS EC2 using Terraform. The infrastructure includes:

  • Custom VPC with public subnet
  • Internet Gateway for public access
  • Security Group with HTTP, HTTPS, and SSH access
  • EC2 instance running Amazon Linux 2
  • Elastic IP for consistent public IP address
  • Automated Apache installation via user data
